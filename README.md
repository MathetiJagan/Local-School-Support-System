# 📚 Local School Support System (SchoolConnect)

A web-based platform that connects **local schools** with **community volunteers** to simplify volunteer recruitment, improve communication, and reduce manual administrative work.

The system enables schools to post volunteer requirements while allowing volunteers to browse available opportunities and apply online through an intuitive interface.

---

## 📌 Project Overview

The **Local School Support System** was developed to bridge the gap between schools in need of volunteers and individuals willing to contribute to educational and community activities.

Instead of relying on manual communication methods, the platform provides a centralized solution where schools can easily publish volunteer opportunities and volunteers can apply with just a few clicks.

This project demonstrates how technology can improve collaboration between educational institutions and local communities while making volunteer management more efficient.

## 🚀 How to Run the Project

### Prerequisites

- XAMPP
- PHP 8+
- MySQL
- Web Browser

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/local-school-support-system.git
```

2. Move the project into the XAMPP `htdocs` folder.

3. Start:

- Apache
- MySQL

4. Import the database

- Open phpMyAdmin
- Create a database

```
school_volunteer_db
```

- Import the provided SQL file.

5. Open your browser

```
http://localhost/local-school-support-system
```
